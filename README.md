
# React

## Environment Prepare

Install `node_modules`:
```bash
yarn
```

### Start project
#### development
```bash
yarn start
```

#### test
```bash
yarn start:test
```

#### production
```bash
yarn start:pro
```

### Build project
#### dev
```bash
yarn build:dev
```

#### test
```bash
yarn build:test
```

#### production
```bash
yarn build
```
